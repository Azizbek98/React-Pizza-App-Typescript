<h4 align="center">Installation Process</h4>

##### Clone repo:

```bash
git clone https://github.com/Azizbek98/React-Pizza-App-Typescript.git
```

##### Install and run:

```bash
npm install
npm run start
```

Build project in production mode:

```bash
npm run build
```

Testing application for checking errors:

```bash
npm run test
```
